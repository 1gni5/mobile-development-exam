## Spécifications
L'application se compose de plusieurs parties:
- Un écran d'acceuil
- Un écran de connexion
- Un écran d'inscription
- Le jeu en lui-même

### Acceuil
__Si le joueur n'est pas encore connecté__: le joueur peut se connecter ou de s'inscrire,
le joueur est ensuite redirigé sur l'activité correspondante.

__Si le joueur est déjà connecté__: le joueur peut choisir un mode de jeu (facile, difficile, expert ou chrono)
il est alors redirigé sur l'activité de jeu, laquelle est chargée différement en fonction du mode. Le joueur peut
également choisir de se déconnecter.
