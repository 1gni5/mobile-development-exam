## Spécifications
L'application se compose de plusieurs parties:
- Un écran d'acceuil/connexion
- Un écran d'inscription
- Le jeu en lui-même

### Acceuil
__Si le joueur n'est pas encore connecté__: le joueur peut se connecter ou s'inscrire. 
Une fois connecté  il est redirigé vers le choix de difficulté de sa partie.

__Si le joueur est déjà connecté__: le joueur est redirigé vers le choix de difficulté de sa partie.

__Une fois sur la page de selection de difficulté__: le joueur peut choisir un mode de jeu (facile, difficile, expert ou chrono)
il est alors redirigé sur l'activité de jeu laquelle est chargée différement en fonction du mode.
Aussi il peut reprendre une potentielle partie précedente qu'il avait mis en pause.  Le joueur peut
également choisir de se déconnecter.


### En Jeu
